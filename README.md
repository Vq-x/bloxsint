# Bloxsint  
First ever roblox osint tool used to find information on a roblox user

### usage
```
git clone https://github.com/phishontop/bloxsint
cd bloxsint/
pip install httpx
pip install colorama
python main.py
```

### Features
- Bloxflip information
- RBXFlip information
- Games played
- RoPro information
- multithreaded
